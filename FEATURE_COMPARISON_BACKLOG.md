# Feature Comparison Backlog: SCM Contractors vs Constructflow

Ordered list of features to add or improve for parity with the reference app (SCM Contractors).  
**Reference:** Feature comparison table (Navigation, Project overview, Goals, Financials, Documents, Logs, Team, Cross-app, Mobile).

## Phase 1 — Project detail (high impact)

| # | Feature | Reference (SCM) | Constructflow gap | Status |
|---|--------|------------------|-------------------|--------|
| 1 | **Home tab / overview** | Home tab with status, budget, contacts, logs, activity | Add "Home" as default tab with widgets | ✅ In progress |
| 2 | **Primary / secondary contact** | Names, roles, Call buttons | Only PM; add primary/secondary + Call | ✅ In progress |
| 3 | **Recent logs on project** | "Recent Logs" snippets + View All | Not on ProjectDetail | ✅ In progress |
| 4 | **Today's activity** | Expenses, Photos, Logs, Files counts | Not on project overview | ✅ In progress |
| 5 | **Earlier activity** | Weekly Hours, Est. Cost | Not on project | ✅ In progress |
| 6 | **Weather** | e.g. 37°F Schererville, wind | DailyLog has weather; no project widget | ✅ In progress |
| 7 | **Project location / maps** | "Open in Google Maps" | Not on project detail | ✅ In progress |
| 8 | **Finish Project** | Yes (button) | Status only; no explicit button | ✅ In progress |
| 9 | **Schedule of Values** | Line items: amount, date, description; Add Value | Change orders only; no SOV view | ✅ In progress |
| 10 | **Outstanding invoices on project** | e.g. -$74,500, #26, due 2/28 | Invoices page only | Planned |
