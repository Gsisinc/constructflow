## üöÄ Deployment Instructions

### Prerequisites
```bash
# Node.js 18+
# npm or pnpm
# Git
```

### Setup
```bash
# Clone repository
gh repo clone Gsisinc/constructflow

# Install dependencies
npm install

# Configure environment
cp .env.example .env
# Add your API keys:
# - VITE_OPENAI_API_KEY
# - VITE_SAM_GOV_API_KEY (optional)
# - Other integration keys

# Run development server
npm run dev

# Build for production
npm run build

# Deploy
npm run preview
```

### Environment Variables Required
```
VITE_OPENAI_API_KEY=sk-...
VITE_SAM_GOV_API_KEY=... (optional)
VITE_BASE44_APP_BASE_URL=...
```
